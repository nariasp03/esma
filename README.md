# esma

Sitio web y sistema de reservas de citas para **esma**, estudio de uñas en Aguascalientes.

## ¿Qué hace?

- **Sitio público** con la presentación de la marca y el catálogo de servicios y precios.
- **(En desarrollo)** Sistema de reservas de citas con gestión de anticipos y panel de administración para la dueña.

## Stack

- [Next.js](https://nextjs.org) 16 — App Router, React Server Components
- React 19 + TypeScript
- Tailwind CSS 4 (tipografía Playfair Display + Inter)
- PostgreSQL (Neon) — para el sistema de reservas
- Desplegado en [Railway](https://railway.app)

## Correr en local

```bash
npm install
npm run dev
```

Luego abre [http://localhost:3000](http://localhost:3000).
